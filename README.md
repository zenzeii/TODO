## Features

- Plus Icon '+'
  - tap '+' to add new task
  - tap & hold '+' to switch to delete mode

- Task Tile
  - tap task tile to edit task 
  - tap & hold task tile to change order

- Others
  - tap 'TODO' to learn about us
  - tap task tile checkbox to mark as done
